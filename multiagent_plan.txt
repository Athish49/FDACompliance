│ Plan to implement                                                                                                                                                                                                  │
│                                                                                                                                                                                                                    │
│ Plan: Multi-Agent Reasoning Layer for FDA Compliance AI                                                                                                                                                            │
│                                                                                                                                                                                                                    │
│ Context                                                                                                                                                                                                            │
│                                                                                                                                                                                                                    │
│ Layers 1-4 are complete: extraction (59,105 chunks), BGE-M3 hybrid embedding (Qdrant), and hybrid retrieval with RRF + reranker. The next step is the multi-agent reasoning layer (Layer 5 from                    │
│ implementation_plan.md) — a LangGraph state machine with 5 specialist agents that takes a natural language compliance question, retrieves relevant CFR chunks, resolves definitions, synthesizes a grounded answer │
│  with citations, verifies it, and detects conflicts.                                                                                                                                                               │
│                                                                                                                                                                                                                    │
│ Key adaptation: No Neo4j. Cross-reference expansion and definition resolution both use Qdrant filtered search instead of graph traversal.                                                                          │
│                                                                                                                                                                                                                    │

│ Implementation Details                                                                                                                                                                                             │
│                                                                                                                                                                                                                    │
│ 1. backend/requirements.txt — Add Dependencies                                                                                                                                                                     │
│                                                                                                                                                                                                                    │
│ langgraph>=0.2.0                                                                                                                                                                                                   │
│ langchain-core>=0.3.0                                                                                                                                                                                              │
│ litellm>=1.40.0                                                                                                                                                                                                    │
│                                                                                                                                                                                                                    │
│ No full langchain needed — only langchain-core (required by langgraph for base types).                                                                                                                             │
│                                                                                                                                                                                                                    │
│ 2. backend/agents/state.py — ComplianceState                                                                                                                                                                       │
│                                                                                                                                                                                                                    │
│ class ComplianceState(TypedDict, total=False):                                                                                                                                                                     │
│     # Input                                                                                                                                                                                                        │
│     query: str                                                                                                                                                                                                     │
│     intent: str  # "compliance_question" | "definition_lookup" | "comparison" | "general"                                                                                                                          │
│                                                                                                                                                                                                                    │
│     # Planner                                                                                                                                                                                                      │
│     sub_questions: list[str]                                                                                                                                                                                       │
│     search_filters: dict  # {part_number, section_number, chapter_number} or empty                                                                                                                                 │
│                                                                                                                                                                                                                    │
│     # Retrieval                                                                                                                                                                                                    │
│     retrieved_chunks: list[dict]                                                                                                                                                                                   │
│     cross_ref_chunks: list[dict]                                                                                                                                                                                   │
│                                                                                                                                                                                                                    │
│     # Definition resolver                                                                                                                                                                                          │
│     definitions_resolved: dict[str, str]  # term → definition text                                                                                                                                                 │
│     definition_chunks: list[dict]                                                                                                                                                                                  │
│                                                                                                                                                                                                                    │
│     # Synthesizer                                                                                                                                                                                                  │
│     draft_answer: str                                                                                                                                                                                              │
│     citations: list[dict]  # [{section, title, text_snippet}]                                                                                                                                                      │
│     confidence_score: float                                                                                                                                                                                        │
│                                                                                                                                                                                                                    │
│     # Verifier                                                                                                                                                                                                     │
│     verification_passed: bool                                                                                                                                                                                      │
│     verification_issues: list[dict]  # [{claim, issue, detail}]                                                                                                                                                    │
│     retry_count: int                                                                                                                                                                                               │
│                                                                                                                                                                                                                    │
│     # Conflict detector                                                                                                                                                                                            │
│     conflicts_detected: bool                                                                                                                                                                                       │
│     conflict_flags: list[dict]  # [{sections, description}]                                                                                                                                                        │
│                                                                                                                                                                                                                    │
│     # Final                                                                                                                                                                                                        │
│     final_response: dict                                                                                                                                                                                           │
│     error: str | None                                                                                                                                                                                              │
│                                                                                                                                                                                                                    │
│ total=False so each node only returns the keys it updates.                                                                                                                                                         │
│                                                                                                                                                                                                                    │
│ 3. backend/agents/llm.py — LiteLLM Wrapper                                                                                                                                                                         │
│                                                                                                                                                                                                                    │
│ - PRIMARY_MODEL = env PRIMARY_MODEL or "ollama/llama3.1:8b"                                                                                                                                                        │
│ - FALLBACK_MODEL_1 = env FALLBACK_MODEL_1 or "groq/llama-3.1-70b-versatile"                                                                                                                                        │
│ - FALLBACK_MODEL_2 = env FALLBACK_MODEL_2 or "openai/gpt-4o"                                                                                                                                                       │
│ - OLLAMA_API_BASE = env OLLAMA_API_BASE or "http://localhost:11434"                                                                                                                                                │
│ - Two functions:                                                                                                                                                                                                   │
│   - llm_completion(messages, max_tokens=1024, temperature=0.1) → str                                                                                                                                               │
│   - llm_completion_json(messages, ...) → str (sets response_format={"type": "json_object"})                                                                                                                        │
│ - Uses litellm.completion(model=PRIMARY, fallbacks=[FB1, FB2])                                                                                                                                                     │
│                                                                                                                                                                                                                    │
│ 4. backend/agents/planner.py — Planner Node                                                                                                                                                                        │
│                                                                                                                                                                                                                    │
│ Input: query, optionally verification_issues (on retry)                                                                                                                                                            │
│ Output: intent, sub_questions, search_filters                                                                                                                                                                      │
│                                                                                                                                                                                                                    │
│ LLM prompt asks for JSON with:                                                                                                                                                                                     │
│ - intent: classify as compliance_question / definition_lookup / comparison / general                                                                                                                               │
│ - sub_questions: 1-4 search queries decomposed from the original question                                                                                                                                          │
│ - search_filters: part_number/section_number if mentioned in query                                                                                                                                                 │
│                                                                                                                                                                                                                    │
│ On retry (when verification_issues exists): appends extra context telling the LLM to adjust sub-questions to find source text for unsupported claims.                                                              │
│                                                                                                                                                                                                                    │
│ 5. backend/agents/retriever_node.py — Retriever Node (NO LLM)                                                                                                                                                      │
│                                                                                                                                                                                                                    │
│ Input: query, sub_questions, search_filters                                                                                                                                                                        │
│ Output: retrieved_chunks, cross_ref_chunks                                                                                                                                                                         │
│                                                                                                                                                                                                                    │
│ Deterministic node — calls CFRRetriever.search() for each sub-question (top_k=10, reranker=True), deduplicates by chunk_id.                                                                                        │
│                                                                                                                                                                                                                    │
│ Cross-reference expansion (replaces Neo4j):                                                                                                                                                                        │
│ - Collect all unique cross_references_internal section numbers from retrieved chunks                                                                                                                               │
│ - For each, do retriever.search(query=original_query, top_k=3, use_reranker=False, filters=SearchFilters(section_number=xref))                                                                                     │
│ - Store separately in cross_ref_chunks                                                                                                                                                                             │
│                                                                                                                                                                                                                    │
│ 6. backend/agents/definition_resolver.py — Definition Resolver                                                                                                                                                     │
│                                                                                                                                                                                                                    │
│ Input: query, retrieved_chunks                                                                                                                                                                                     │
│ Output: definitions_resolved, definition_chunks                                                                                                                                                                    │
│                                                                                                                                                                                                                    │
│ Phase 1 (LLM): Identify regulated terms needing formal definitions from the retrieved context. Returns JSON {"terms": ["term1", ...]}.                                                                             │
│                                                                                                                                                                                                                    │
│ Phase 2 (Qdrant): For each term, search with SearchFilters(chunk_type="definition") and query "definition of {term}". Match on the defines payload field (exact case-insensitive match first, fuzzy fallback to    │
│ best scoring result).                                                                                                                                                                                              │
│                                                                                                                                                                                                                    │
│ 7. backend/agents/synthesizer.py — Synthesizer Node                                                                                                                                                                │
│                                                                                                                                                                                                                    │
│ Input: query, retrieved_chunks, cross_ref_chunks, definitions_resolved                                                                                                                                             │
│ Output: draft_answer, citations, confidence_score                                                                                                                                                                  │
│                                                                                                                                                                                                                    │
│ Builds context from:                                                                                                                                                                                               │
│ - Definitions block (resolved terms)                                                                                                                                                                               │
│ - Primary chunks (top 8, with cfr_citation, preamble, text, overflow)                                                                                                                                              │
│ - Cross-ref chunks (top 4)                                                                                                                                                                                         │
│                                                                                                                                                                                                                    │
│ LLM returns JSON with answer (inline [21 CFR X.Y] citations), citations list, and confidence score (0-1 based on coverage guidelines). max_tokens=2048 for this call.                                              │
│                                                                                                                                                                                                                    │
│ 8. backend/agents/verifier.py — Verifier Node                                                                                                                                                                      │
│                                                                                                                                                                                                                    │
│ Input: draft_answer, retrieved_chunks, cross_ref_chunks, retry_count                                                                                                                                               │
│ Output: verification_passed, verification_issues, retry_count                                                                                                                                                      │
│                                                                                                                                                                                                                    │
│ LLM checks every factual claim against source chunks. Returns JSON with verification_passed (bool) and issues list. Strict mode: specific numbers, thresholds, section references must appear verbatim in source.  │
│                                                                                                                                                                                                                    │
│ Increments retry_count when issues are found.                                                                                                                                                                      │
│                                                                                                                                                                                                                    │
│ 9. backend/agents/conflict_detector.py — Conflict Detector + Final Assembly                                                                                                                                        │
│                                                                                                                                                                                                                    │
│ Input: Full state                                                                                                                                                                                                  │
│ Output: conflicts_detected, conflict_flags, final_response                                                                                                                                                         │
│                                                                                                                                                                                                                    │
│ LLM checks retrieved sections for contradictions (general rule vs exception, conditional applicability, etc). Returns JSON with flags.                                                                             │
│                                                                                                                                                                                                                    │
│ Also assembles final_response dict:                                                                                                                                                                                │
│ {                                                                                                                                                                                                                  │
│     "answer": state["draft_answer"],                                                                                                                                                                               │
│     "citations": state["citations"],                                                                                                                                                                               │
│     "confidence_score": state["confidence_score"],                                                                                                                                                                 │
│     "conflicts_detected": bool,                                                                                                                                                                                    │
│     "conflict_details": [...],                                                                                                                                                                                     │
│     "disclaimer": "This is for informational purposes only...",                                                                                                                                                    │
│     "retrieved_sections": [...],  # unique section numbers                                                                                                                                                         │
│     "verification_passed": bool,                                                                                                                                                                                   │
│ }                                                                                                                                                                                                                  │
│                                                                                                                                                                                                                    │
│ 10. backend/agents/graph.py — LangGraph Wiring                                                                                                                                                                     │
│                                                                                                                                                                                                                    │
│ graph = StateGraph(ComplianceState)                                                                                                                                                                                │
│ graph.add_node("planner", planner_node)                                                                                                                                                                            │
│ graph.add_node("retriever", retriever_node)                                                                                                                                                                        │
│ graph.add_node("definition_resolver", definition_resolver_node)                                                                                                                                                    │
│ graph.add_node("synthesizer", synthesizer_node)                                                                                                                                                                    │
│ graph.add_node("verifier", verifier_node)                                                                                                                                                                          │
│ graph.add_node("conflict_detector", conflict_detector_node)                                                                                                                                                        │
│                                                                                                                                                                                                                    │
│ graph.set_entry_point("planner")                                                                                                                                                                                   │
│ graph.add_edge("planner", "retriever")                                                                                                                                                                             │
│ graph.add_edge("retriever", "definition_resolver")                                                                                                                                                                 │
│ graph.add_edge("definition_resolver", "synthesizer")                                                                                                                                                               │
│ graph.add_edge("synthesizer", "verifier")                                                                                                                                                                          │
│                                                                                                                                                                                                                    │
│ # Conditional: retry or proceed                                                                                                                                                                                    │
│ graph.add_conditional_edges("verifier", should_retry, {                                                                                                                                                            │
│     "planner": "planner",             # retry (issues + retry_count < 2)                                                                                                                                           │
│     "conflict_detector": "conflict_detector",  # proceed (ok or max retries)                                                                                                                                       │
│ })                                                                                                                                                                                                                 │
│ graph.add_edge("conflict_detector", END)                                                                                                                                                                           │
│                                                                                                                                                                                                                    │
│ query_graph = graph.compile()                                                                                                                                                                                      │
│                                                                                                                                                                                                                    │
│ 11. backend/main.py — Add Query Endpoint                                                                                                                                                                           │
│                                                                                                                                                                                                                    │
│ @app.post("/api/query", tags=["query"])                                                                                                                                                                            │
│ def query_compliance(request: QueryRequest):                                                                                                                                                                       │
│     from backend.agents.graph import query_graph                                                                                                                                                                   │
│     result = query_graph.invoke({"query": request.question, "retry_count": 0})                                                                                                                                     │
│     return result["final_response"]                                                                                                                                                                                │
│                                                                                                                                                                                                                    │
│ QueryRequest: question: str (min_length=5)                                                                                                                                                                         │
│ QueryResponse: answer, citations, confidence_score, conflicts_detected, conflict_details, disclaimer, retrieved_sections, verification_passed                                                                      │
│                                                                                                                                                                                                                    │
│ Error Handling Strategy                                                                                                                                                                                            │
│                                                                                                                                                                                                                    │
│ Every json.loads() on LLM output wrapped in try/except. On parse failure:                                                                                                                                          │
│ 1. Retry LLM once with "Your response was not valid JSON. Return ONLY valid JSON."                                                                                                                                 │
│ 2. If still fails, set error in state and return a graceful error response.                                                                                                                                        │
